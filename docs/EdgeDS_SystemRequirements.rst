
****************************
 Edge DS System Requirements
*****************************

Recommended Hardware Configuration
**********************************
1GB Ram or Greater - Highly dependent on the number of streams
Dual Core CPU X86 or ARM
Persistent Storage, SSD Preferred
Historian Files 128 MB Required
Historical Storage - TBD

Supported Operating Systems
****************************
Ubuntu 16.04
Windows 10, 2016 
Windows IoT Core 
Debian 9 - ARM

Prerequisites
****************

The Edge DS requires the following software to be installed prior to installing Edge DS. 

Linux - libunwind8 

The Edge DS platform specific packages contains all files necessary to run other than those listed above. 
We also have platform independent packages which will require .netcore 2.0 to be installed prior to installing Edge DS.
